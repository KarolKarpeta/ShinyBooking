A) HTML Structure - top

The template is made with responsive layout with Bootstrap4 framework. All the content is within section. All the part of the template is divided with section. Main infrastructure is same into the all section. All contents are rounded with div, id and class. Most of the class used according to Bootstrap. Here is the general structure.
